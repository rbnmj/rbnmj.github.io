EcoRSS

EcoRSS is a growing project that processes and aggregates RSS feeds from various ecology related journals. The project includes a build script that fetches, processes, and organizes articles into a JSON file, which is then deployed to GitHub Pages. At this point the backbone is working, but there is absolutely room for expanded functionality and scale. Visit the Repository on GitHub if you are interested in contributing.

Features

  • Lightweight, up-to-date RSS feed of ecology journals.
  • No bias in article selection due to algorithms.
  • Completely open source service without monetization concerns.
  • Open to contributions and improvements.

Sports Facility Development Planning

Planning for communal sports facility development often involves extensive citizen surveys and evaluations. Crawling through endless Excel sheets and PDFs can be a tedious task. This project aims to automate the evaluation process of surveys for sports facility development concepts (Sportstättenentwicklungskonzept). The foundation of this project are the guidelines published by the German Federal Institute for Sport Science (BISp). The goal is not only to calculate the total need of sports facility but also consider verbal feedback through LLM-supported evaluation. This is realized in Python using sklearn for machine learning implementation and accessing the OpenAI API for natural language processing.
Feel free to reach out if you're interested in using or collaborating on this project. Just shoot me an email or connect with me on LinkedIn.

iPM-ODE

iPM-ODE (interactive Popuation Models with Ordinary Differential Equations) is designed to help life science students understand the dynamics of ecological models and how parameter effect simulation outcomes. Created for a theoretical ecology course, it provides an accessible way to explore basic population dynamics without the frustration that may come from solely relying on differential equations. This approach is particularly beneficial for students with limited mathematical backgrounds, as it uses descriptive tools to foster a deeper understanding of key concepts.
iPM-ODE is an object-oriented Python tool featuring a user interface built using Matplotlib Widgets.

Features

  • Interactive parameter sliders for real-time model adjustments.
  • Predefined ecological models for quick exploration.
  • Open source approach for advanced users to define and add their own models.
  • Visualization of population dynamics through fundamental models (e.g. Lotka-Volterra, Rosenzweig-Macarthur).
